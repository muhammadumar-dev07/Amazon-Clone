# Amazon UI Clone 🛒

A front-end clone of the Amazon website built using pure HTML, CSS, and JavaScript. This was one of my first major UI projects and I built it to practice responsive design, DOM manipulation, and real-world layout structure.

## 🚀 Features

- Responsive navbar with search bar and cart icon
- Mobile hamburger menu with slide-in animation
- Hero banner section
- Shop by Category section with 4 categories
- 12 product cards with real product names, prices, ratings and badges
- Live search filter — filters products as you type
- Add to Cart with counter and button feedback
- Footer with links and social icons
- Fully responsive on mobile, tablet, laptop and large screens

---

## 🛠️ Technologies Used

- HTML5
- CSS3 (Flexbox, Grid, Media Queries)
- JavaScript (DOM Manipulation, Events)
- Font Awesome (icons)

## ▶️ How to Run

1. Download or clone this repository
2. Open the `amazon-clone` folder
3. Double click `index.html` to open it in your browser
4. No installation or setup needed

## 📱 Responsive Breakpoints

| Screen Size | Layout |
|---|---|
| Large (1600px+) | 5 product columns |
| Desktop (1200px) | 3 product columns |
| Tablet (900px) | 2 columns, hamburger menu |
| Mobile (600px) | 2 columns, compact cards |
| Small Mobile (400px) | 1 column |
